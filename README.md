# Prerequisites

- Node.js [Install Here](https://nodejs.org/en/download)
  - Verify by running in terminal
    ```bash
    node -v
    ```
- npm (preferred) or equivalent Javascript package manager
  - Verify by running in terminal
    ```bash
    npm -v
    ```

# Setup Enviornment

- Packages to install:
  - express (used to create a server)
  ```bash
  npm install express
  ```
  - mysql2 (used to query the database with Javascript)
  ```bash
  npm install mysql2
  ```
  - hbs (used to template - can render page dynamically based on data sent from server)
  ```bash
  npm install hbs
  ```
